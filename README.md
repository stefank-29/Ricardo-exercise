# Installation

Clone repository:

```
git clone https://github.com/stefank-29/Ricardo-exercise.git
```

### Run project:

```
cd Ricardo-exercise/
```

Create .env.local file with NEXT_PUBLIC_API_TOKEN key for Ricardo api token

```

npm install

npm run dev

```

Open in browser at address:

```

http://localhost:3000/

```

### Built with:

-   React (Next.js)
